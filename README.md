# Enterprise Network VLAN Lab

## Overview
Built a segmented network using VLANs and inter-VLAN routing in Cisco Packet Tracer.

## Technologies
- Cisco Packet Tracer
- VLANs
- 802.1Q Trunking
- Router-on-a-Stick

## Features
- VLAN segmentation (HR, IT, Finance)
- Inter-VLAN routing
- Packet simulation testing

## Testing
- ICMP ping across VLANs
- Packet flow analysis in simulation mode
